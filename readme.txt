Estimate post time reading automatically

== Description ==
The best plugin to estimate how many minutes you need to read the post. You can customize in which posts will be shown.

== Installation ==
1. Install&Activate the plugin.
2. Add the Timer Post Widget to the sidebar and configure it.
3. Check the box in the posts where you want the estimation to be showed.
